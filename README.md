### EVM Token Tools [Rust]

1. install rustup 

    https://www.rust-lang.org/tools/install

2. add to .env 

    (see .env.template)

3. run these scripts 

    cargo run --bin SCRIPT_NAME





#### Sample .env file 

```

RPC_URL="https://mainnet.g.alchemy.com/v2/xxxx"
PRIVATE_KEY="65xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxd6"


CHAIN_ID=5



```
